Класс Rational

Длинное Рациональное число. Подробнее...

#include <rational.hpp>

Подробное описание

Длинное Рациональное число.

Числитель имеет тип Integer, знаменатель имеет тип Natural. Для этого класса перегружены основные арифметические операторы, что позволяет красиво записывать различные вычисления. Также для этого класса перегружены операторы ввода/вывода.
